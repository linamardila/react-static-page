# Alice's Adventures in Wonderland - React Static Page

A simple React application that displays the classic story of Alice in Wonderland using reusable components.

## Project Description

This project demonstrates the fundamentals of React by creating a static webpage that tells the story of Alice's Adventures in Wonderland. The application is built using three separate components (Header, Content, and Footer) that are imported and rendered in the main App component.

## Features

- **Component-based architecture**: Organized into Header, Content, and Footer components
- **Responsive design**: Clean and readable layout that works on different screen sizes
- **Fairytale styling**: Soft, dreamy color scheme with elegant typography
- **Story presentation**: Clear narrative structure with moral lesson
